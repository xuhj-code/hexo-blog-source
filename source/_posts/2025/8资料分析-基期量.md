# 基期量 · 核心速览
| 基期量概念          | 基期计算公式                 | 同比 / 环比         |
| -------------- | ---------------------- | --------------- |
| 基期是对比参照期，与现期相对 | 基期=现期−增长量 / 现期÷(1+增长率) | 同比看年份，环比看相邻最小时间 |

| 高减低加      | 化除为乘 | 基期比较 |                  |             |
| --------- | ---- | ---- | ---------------- | ----------- |
| 增长用减，减少用加 |      | r    | ≤5% 时可近似 a×(1−r) | 现期大+增速小→基期大 |

| 基期和差           | 累计值        | 进出口      |
| -------------- | ---------- | -------- |
| 整体/比重求基期，差值需反推 | 季度=累计−上期累计 | 顺差=出口−进口 |

# 基期量 · 小结总览

| 知识点     | 核心内容                                                 | 考试重点 / 易混淆点              | 难度系数  |
| ------- | ---------------------------------------------------- | ------------------------ | ----- |
| 基期量概念   | 作为对比参照的时期量，与现期量相对                                    | 区分基期量与现期量的时间关系           | ⭐⭐ |
| 基期量计算公式 | 公式1：基期=现期-增长量<br>公式2：基期=增长量/增长率<br>公式3：基期=现期/(1+增长率) | 公式选择依据题目给定条件             | ⭐⭐⭐ |
| 同比环比区分  | 同比：与上年同期相比（看年份）<br>环比：与相邻上期相比（看最小时间单位）               | 日期环比计算易错（如5月12日环比是5月11日） | ⭐⭐⭐⭐ |
| 高减低加原则  | 现期比基期增长→基期=现期-增长量<br>现期比基期减少→基期=现期+减少量               | 多退少补原理类比                 | ⭐⭐⭐ |
| 基期和差计算  | 整体=部分/比重→再求基期<br>基期差=现期差/(1+增长率差)                    | 注意材料中的总分结构陷阱             | ⭐⭐⭐⭐ |
| 化除为乘技巧  | 当 |r|≤5% 时：a/(1+r)≈a×(1-r)                           | 注意反例情况（选项首位相同+4%≤|r|≤5%） | ⭐⭐⭐⭐ |
| 基期比较方法  | 现期量大+增长率小→基期量大                                       | 先排除再计算，数据量级优先比较策略        | ⭐⭐⭐ |
| 进出口概念   | 顺差=出口-进口>0<br>逆差=进口-出口>0                             | 统计口径一致性检查                | ⭐⭐ |
| 累计值计算   | 季度值=截止月累计-上月累计<br>（如Q3=1-9月-1-6月）                    | 注意增长率匹配问题                | ⭐⭐⭐⭐ |
| 棉花专项考点  | 面积/单产/产量三要素关系<br>新疆 vs 黄河流域 vs 长江流域对比                | 主体定位准确性要求高               | ⭐⭐⭐ |
