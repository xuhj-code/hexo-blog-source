# 数量规律 ·面·线·点· 核心速览
| 对称轴优先                | 面的考法         | 线的考法           |
| -------------------- | ------------ | -------------- |
| **对称轴是第一优先级**，黑白块先看轴 | 白色封闭才算面，缺口不算 | 拐弯算多条线，先分直 / 曲 |
| 轴对称 / 中心对称直接秒        | 面数可横向 / 纵向相加 | 两种线常考相加 / 倍数   |
| 对称成立，其它规律不用看         | 最大面 / 相同面是高频 | 平行线按“同方向一组”    |

| 点的考法          | 曲线快速分流           | 相邻比较            |
| ------------- | ---------------- | --------------- |
| 只数**交点**，端点不算 | 圆 → 切点；非圆 → 曲直交点 | 有相同就比，不同是突破口    |
| 多端点先想笔画       | 部分曲线 ≠ 全曲线       | 动的是变量，不动是关键     |
| 点常配合线一起考      | 曲线强于面特征          | 比位置 / 比数量 / 比存在 |

| 优先级总纲                       |              |
| --------------------------- | ------------ |
| **对称轴 ＞ 线 ＞ 面 ＞ 点**，先抓最显眼特征 | **不要一上来就数面** |
| 曲线特征 ＞ 面特征                  | 复杂题先排明显错误选项  |
| 看不出就回到“对称 + 相邻比较”           |              |

# 数量规律 ·面·线·点· 总览速览表

| 知识点       | 核心内容                       | 考试重点 / 易混淆点            | 难度系数 |
| --------- | -------------------------- | ---------------------- | ---- |
| 对称轴优先级    | 对称轴在黑白块题目中优先级最高，无需考虑其他规律   | 对称轴是“大哥”，直接验证对称性       | ⭐⭐⭐  |
| 中心对称识别    | 通过分割图形找相同部分（如左上角与右下角对称）    | 旋转 180 度后图形重合即为中心对称    | ⭐⭐   |
| 面的数量      | 白色封闭区域算作面，黑色区域不算           | 缺口图形不算面，需满足封闭和白色两个条件   | ⭐⭐   |
| 面的运算      | 九宫格中面的数量可能考横向或纵向相加         | 二 + 三 = 五或每行总和相等       | ⭐⭐⭐⭐ |
| 面的细化（所有面） | 每个面的形状相同（如全三角形、全四边形）       | 单数三角形、三角形与四边形运算、三角形递增  | ⭐⭐⭐⭐ |
| 大小面       | 最大面或最小面的形状、对称性、占比（如占 1/2）  | 最大面与外框形状一致是高频考点        | ⭐⭐⭐  |
| 相同面       | 大小和形状完全相同的面                | 易与对称性混淆，需排除对称轴干扰       | ⭐⭐   |
| 线的数量      | 直线和曲线的区分（拐弯处算多条线）          | 多边形外框线条数可能考拆分思想        | ⭐⭐⭐  |
| 线的运算      | 两种线（如曲线与直线）的数量关系（相减、倍数、相加） | 相加关系最难识别               | ⭐⭐⭐⭐ |
| 平行线       | 斜线或相似三角形暗示平行线组数            | 相同方向为一组，斜线易出三组以上       | ⭐⭐⭐  |
| 点的数量      | 焦点（线线相交）才算点，端点不算           | 多端点图形优先考虑笔画再数点         | ⭐⭐   |
| 点的细化      | 切点（圆与线相切）、框点（框内/框外交点）、曲直交点 | 圆优先考虑切点，非圆考虑曲直交点       | ⭐⭐⭐⭐ |
| 曲线考点区分    | 部分曲线（曲直性） vs 全曲线（数量规律）     | 圆与非圆的分类决定考点（切点 / 曲直交点） | ⭐⭐⭐⭐ |
| 相邻比较思想    | 两幅图之间有相同部分可比较（如移动、增加元素）    | 不一样的部分可能有规律或无规律        | ⭐⭐⭐  |
| 数量规律优先级   | 对称轴 ＞ 线 ＞ 面 ＞ 点，曲线特征强于面    | 避免一上来就数面，先抓明显特征        | ⭐⭐⭐⭐ |
