# 速算技巧 ·加·减·乘· 知识小结总览

| 知识点      | 核心内容                                                                 | 考试重点 / 易混淆点                                    | 难度系数 |
| -------- | -------------------------------------------------------------------- | ---------------------------------------------- | ---- |
| 尾数法      | 适用于加减精算，选项尾数不同时使用，如 893+99+33 直接计算尾数 3+9=12→2+3=5 对应选项尾数 5           | 精确度需一致（个位对个位）                                  | ⭐⭐   |
| 高位叠加     | 多位数相加时从高位算起，前两位精算（两加），第三位估算（一估），如 10 个 1000=1 万 + 百位 16+4=20 → 1.2 万 | 数据有效性优先（千位＞百位），选项接近时需多算一位                      | ⭐⭐⭐  |
| 划线减法     | 避免借位的减法技巧，如 714−688 拆分为 (700−688)+(714−700)=12+14=26                 | 临界值法适用于连续借位，普通数值用「一二二一」画法（如 814−634=180+2=182） | ⭐⭐   |
| 乘法技巧     | 高位乘起叠加（65×7=60×7+5×7），拆分法（53×21=53×20+53×1），口诀法（前前后后里里外外）            | 两位数乘两位数优先用口诀法，如 56×36 → 2016                   | ⭐⭐⭐⭐ |
| 特殊数字运算   | ×1.5=原数+半值；×1.1≈错位相加；×0.9≈错位相减                                       | 百分数转换：1.1≈1/0.9，0.9≈1/1.1                      | ⭐⭐⭐  |
| 百分数互化    | 常用分数对应百分数：1/6≈16.7%，1/7≈14.3%，1/9≈11.1%，1/11≈9.1%                    | 记忆规律：6↔16，7↔14；加和为 20 的分数快速换算                  | ⭐⭐⭐⭐ |
| 包子法（画一法） | 选项接近时使用，如 145÷142≈100%+3/142≈102%                                    | 分母截两位当百分号；50%附近配合画半法                           | ⭐⭐⭐⭐ |

# 速算技巧 ·加·减·乘· 知识小结总览

| 知识点     | 核心内容                   | 考试重点 / 易混淆点                        | 难度系数 |
| ------- | ---------------------- | ---------------------------------- | ---- |
| 速算技巧-除法 | 选项决定截位方法：差距大截两位，差距小截三位 | 首位相同第二位差 > 首位为差距大                  | ⭐⭐⭐  |
| 分数比较    | 一大一小直接看（分子大分母小则值大）     | 同大同小看倍数关系：分子快则分子大的值大               | ⭐⭐   |
| 量级判断    | 百分号量级用包子法，数值量级注意小数点位移  | 亿 = 10⁸，万 = 10⁴，统一量级再比较            | ⭐⭐⭐⭐ |
| 等比例修正   | 按分子分母倍数关系同比例增减         | 优先调整大数，如 98→100 加 2，252 按 2.5 倍加 5 | ⭐⭐⭐⭐ |
| 特殊题型处理  | 整数左右邻居（如 59/61）需精算     | 首位 + 第二位相同需截四位，临界值附近默认差距小          | ⭐⭐⭐⭐ |
| 多步除法    | 分子分母都截位，截完约分           | 注意只截分母与都截的区别                       | ⭐⭐⭐  |
| 实战技巧    | 结合选项反推商值，少列式子          | 选项会提示商几，如 70÷15 看 36/46/56/76 选项   | ⭐⭐⭐  |
| 分数排序    | 先分析分子 / 分母极值确定候选范围     | 材料问法转化（如最小比重 → 最大倒数）               | ⭐⭐⭐⭐ |
