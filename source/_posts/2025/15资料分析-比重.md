# 比重 · 核心速览
| 现期比重             | 比重比较         | 比重差          |
| ---------------- | ------------ | ------------ |
| 部分量 ÷ 总量，先判“谁占谁” | 同总体直接比部分量    | 同总体：部分差 ÷ 总体 |
| “A占B”≠“B中A”      | 不同总体先算比重再比   | 不同总体：先比重后相减  |
| 单位、口径先统一         | 倍数 / 直除 / 估算 | 方向看分子，大小看量级  |

| 基期比重             | 两期比重变化      | 多期 / 图形     |
| ---------------- | ----------- | ----------- |
| 现期比重×(1+b)/(1+a) | a＞b 升，a＜b 降 | 饼图先看1/2、1/3 |
| 增速带符号            | 逆向可反推       | 排序按顺时针      |
| 先判断＞1或＜1         | 差值公式三步法     | 量级异常必核查     |

| 比重连乘    | 利润率 / 产销率   | 高频陷阱   |
| ------- | ----------- | ------ |
| 小占中×中占大 | 利润/收入；销量/产量 | 占比≠增长率 |
| 过渡消元    | 区分统计口径      | 营业额≠数量 |
| 链式比例    | 零增长特例       | 单位不统一  |

# 比重 · 小结总览

| 知识点        | 核心内容                                     | 考试重点 / 易混淆点                            | 难度系数 |
| ---------- | ---------------------------------------- | -------------------------------------- | ---- |
| 现期比重计算     | 部分量 / 总量，注意“占”前后的部分与总体关系                 | 区分“A占B”与“B中A”的分子分母差异                   | ⭐⭐   |
| 现期比重比较     | 分数比较法（横看倍数 / 竖看直除），估算范围优先                | 量级陷阱，倒数比较法反向应用                         | ⭐⭐⭐  |
| 特定比重问题     | 总体 × 特定比重 = 阈值，部分 ＞ 阈值即达标                | 多部分求和时的高位叠加与凑整技巧                       | ⭐⭐   |
| 现期比重差      | 相同总体：部分差 / 总体；不同总体：分别求比重再相减              | 选项差距大时估算范围优先                           | ⭐⭐⭐  |
| 计算总体 / 部分量 | 总体 = 部分 / 比重；部分差 = 总体 × 比重差              | 单位统一（万亿 / 亿转换）                         | ⭐⭐   |
| 饼图分析       | 构图原则：12点方向顺时针排序，主体优先验证                   | 非常规排序需看比例（1/2、1/3等明显特征）                | ⭐    |
| 比例类比       | 同总体中部分量倍数 = 比重倍数（a/b = ka/kb）            | 分子给、分母未给时的快速推导                         | ⭐⭐⭐  |
| 比重连乘       | 小占中 × 中占大 = 小占大（过渡消元）                    | 链式比例中的量级一致性检查                          | ⭐⭐   |
| 基期比重公式     | 现期比重 × (1+b) / (1+a)                     | 交叉对应增长率正负号处理（如 a = -10% 写为 0.9）        | ⭐⭐⭐⭐ |
| 基期比重速算     | 分析型：算前看后（现期比重 × ＞1 / ＜1）；计算型：两位约分 / 等比修正 | 选项差距极小时需三位精算                           | ⭐⭐⭐⭐ |
| 两期比重比较     | a ＞ b 则升，a ＜ b 则降（带符号比较）                 | 逆向应用：比重升 → a ＞ b，降 → a ＜ b             | ⭐⭐⭐  |
| 多年比重趋势     | 定首尾看内部变化，结合增速与比例关系                       | 混合比例（如二季度环比需用混合思维）                     | ⭐⭐⭐⭐ |
| 两期比重差计算    | 现期比重 × (a - b) / (1 + a)                 | a - b 定方向；公式三步法：a-b → 大 a/b → 1+a，估算优先 | ⭐⭐⭐  |
| 利润率特殊规则    | 资料分析：利润 / 收入；数量关系：利润 / 成本                | 行业差异（宏观 vs 微观统计口径）                     | ⭐⭐   |
| 亏损面 / 产销率  | 亏损面 = 亏损企业数 / 总企业数；产销率 = 销量 / 产量         | 避免营业额与数量混淆                             | ⭐    |
