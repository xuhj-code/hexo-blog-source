# 平均数 · 核心速览
| 现期平均数            | 平均数比较      | 双重平均数    |
| ---------------- | ---------- | -------- |
| 后除前原则（如人均=收入/人数） | a>b 则平均数上升 | 月均=年均÷12 |
| 单位面积=产量/面积       | 可用增长率快速判断  | 优先除整数组   |
| 注意量级与单位转换        | 常识范围辅助判断   | 万亿/亿需统一  |

| 平均数增长量            | 平均数增长率      | 连续多年比较  |
| ----------------- | ----------- | ------- |
| (a-b)/(1+a)×(A/B) | (a-b)/(1+b) | 定首尾看趋势  |
| 增长率交叉对应           | 区分比重差公式     | 核验时间段首年 |
| 正负号先判方向           | 分母不同勿混用     | 年份陷阱必查  |

| 削峰填谷法  | 高位叠加法  | 速算适用前提 |
| ------ | ------ | ------ |
| 基准线求均值 | 高位逐级相加 | 数据差异要小 |
| 适合差距不大 | 末位取中值  | 仅用于估算  |
| 快而不精算  | 排除法优先  | 精算再回代  |

# 平均数 · 小结总览

| 知识点      | 核心内容                                     | 考试重点 / 易混淆点                 | 难度系数 |
| -------- | ---------------------------------------- | --------------------------- | ---- |
| 现期平均数计算  | 后除前原则（如：人均收入 = 收入 / 人数；单位面积产量 = 产量 / 面积） | 注意量级转换（如公顷 → 亩需 ×15）        | ⭐⭐   |
| 特定平均数比较  | 通过增长率快速判断（a > b 则上升）                     | 电影票单价比较时忽略量级（常识判断 40 元合理范围） | ⭐⭐   |
| 双重平均数    | 月均 = 年均 ÷ 12，优先除整数组（如时间单位）               | 单位统一（如万亿 / 亿需约分）            | ⭐⭐⭐  |
| 平均数增长量公式 | = (a − b) / (1 + a) × (A / B)            | 注意分子分母增长率交叉对应，正负判断优先排除选项    | ⭐⭐⭐  |
| 平均数增长率公式 | = (a − b) / (1 + b)                      | 与比重差公式区分（分母不同）              | ⭐⭐   |
| 连续多年比较   | 定首尾法：首年与末年升降决定趋势                         | 需验证时间段第一年（如 2018 需对比 2017）  | ⭐⭐⭐  |
| 削峰填谷法    | 基准线法计算多数据平均值（如取 200 为基准）                 | 仅适用于数据差异小的场景                | ⭐⭐   |
| 高位叠加法    | 万位 / 千位逐级相加估算总和（如 11 个月数据相加）             | 末位按中间值估算（如都取 5）             | ⭐⭐   |
