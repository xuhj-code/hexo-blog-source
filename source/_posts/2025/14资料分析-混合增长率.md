# 混合增长率 · 核心速览
| 混合增长率概念                  | 混合增长率判定                            | 固定搭配案例                                              |
| ----------------------- | ------------------------------------ | ---------------------------------------------------- |
| 部分增长率混合成整体增长率，必须存在“部分 + 部分 = 整体” | 明显有部分混合到整体，注意与乘积增长率区分（加和 vs 乘积） | 进出口 = 进口 + 出口；全国 = 城镇 + 乡村；邮电 = 邮政 + 电信 |

| 混合居中原则                                   | 偏向原理                 | 线段法精算                                  |
| ----------------------------------------- | ------------------------ | ---------------------------------------- |
| 整体增长率介于部分增长率之间；负增长率必须带符号比较 | 偏向基期量大的部分，现期量代替基期量（差异 < 20%） | 距离 = 部分与整体增长率差，距离与量成反比；增长率差大时用基期量计算 |

| 差值转化技巧         | 累计增速规律               | 多者混合处理 / 返利识别               |
| -------------- | ------------------ | ------------------ |
| 顺差 = 出口 - 进口 → 出口 = 进口 + 顺差，转化为加和关系应用混合原理 | 当月增速 > 累计增速 → 累计增速上升；时间维度混合（1-11月 + 12月 = 全年） | 多部分混合保持大小居中原则，现期量接近或增长率差异 > 20% 精算截三位 |

# 混合增长率 · 小结总览

| 知识点     | 核心内容                        | 考试重点 / 易混淆点              | 难度系数  |
| ------- | --------------------------- | ------------------------ | ----- |
| 混合增长率概念 | 部分增长率混合成整体增长率的关系            | 必须存在部分 + 部分 = 整体的量关系     | ⭐⭐ |
| 混合增长率判定 | 明显有部分混合到整体的关系               | 求增长率与乘积增长率的区别（加和 vs 乘积）  | ⭐⭐⭐ |
| 固定搭配案例  | 进出口=进口+出口；全国=城镇+乡村；邮电=邮政+电信 | 累计增长率=混合增长率的特殊考法         | ⭐⭐ |
| 混合居中原则  | 整体增长率介于部分增长率之间              | 负增长率必须带符号比较              | ⭐⭐⭐ |
| 偏向原理    | 混合增长率偏向基期量大的部分              | 现期量代替基期量的适用条件（差异<20%）    | ⭐⭐⭐⭐ |
| 线段法精算   | 距离=部分与整体增长率差，距离与量成反比        | 增长率差异大时必须用基期量计算          | ⭐⭐⭐⭐ |
| 差值转化技巧  | 顺差=出口-进口 → 出口=进口+顺差         | 必须转化为加和关系才能应用混合原理        | ⭐⭐⭐⭐ |
| 累计增速规律  | 当月增速>累计增速 → 累计增速上升          | 时间维度混合（1-11月 + 12月 = 全年） | ⭐⭐⭐ |
| 多者混合处理  | 多个部分混合时保持大小居中原则             | 优先两两等量混合简化计算             | ⭐⭐⭐ |
| 返利识别    | 现期量接近或增长率差异>20%             | 需精算截三位处理微小基期量            | ⭐⭐⭐⭐⭐ |
