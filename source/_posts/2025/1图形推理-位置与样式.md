# 位置规律 · 核心速览

|          位置规律             |         相邻比较         |          旋转翻转          |
| ---------------------------- | ---------------------- | ------------------------- |
| 平移方向 + 步数，16 宫格内外分开 | 找不动元素，其余只换位置  | 顺逆时针角度 + 翻转轴     |

|          九宫格规律            |         16 宫格        |        25 宫格与蜂窝        |
| ---------------------------- | ---------------------- | ------------------------- |
| 等差 / 恒定 / 周期，横向优先  | 内外分离，重点内部递增  | 斜向整体运动，大面积定位 |

|         非宫格难点             |         解题技巧        |          心理建设          |
| ---------------------------- | ---------------------- | ------------------------- |
| 隐藏线条、拆分运动          | 大面积 → 个数 → 对比 | 难题跳过，70% 即高分     |

# 样式规律 · 核心速览

|             样式规律                   |           黑白运算         |        标记考点          |
| ------------------------------------- | --------------------------- | -------------------------------- |
| 部分相同元素做运算，加减 / 求同 / 求异（高频求异） | 外框相同看黑块变化，相同区域秒杀优先 | 点 / 线 / 角 / 面，关注标记位置与关系 |

|             动态样式                   |           拆分思维         |        金字塔运算          |
| ------------------------------------- | --------------------------- | ---------------------- |
| 先旋转再运算，一三同转二 / 二三同转一      | 不同区域不同规则，上下或内外分开  | 下层两元素决定上层一元素   |

# 属性规律 · 核心速览
| 对称性                       | 曲直性                    | 开闭性                        |
| ------------------------- | ---------------------- | -------------------------- |
| 看对称轴方向与数量，区分轴 / 中心 / 轴+中心 | 用 1/2、2/3 比例定位，判断是否含曲线 | 生活化优先：全开 / 半开 / 全闭（点连接也算开） |

| 图形间关系                 | 黑白块专项                | 最新趋势              |
| --------------------- | -------------------- | ----------------- |
| 简单看相交点线面，复杂找共同部分 + 差异 | 顺序：对称性 ＞ 位置规律 ＞ 黑白运算 | 不看整体，看未相交部分或相交线位置 |

# 位置规律 · 总览速览表

| 知识点       | 核心内容                                                | 考试重点 / 易混淆点                 | 难度系数 |
| --------- | --------------------------------------------------- | --------------------------- | ---- |
| 位置规律      | 通过元素个数相同或部分重叠判定位置规律，优先考虑平移方向（横向 / 纵向 / 斜向）和步数       | 16 宫格内外分开看是高频考点，注意内部相等或递增规律 | ⭐⭐⭐⭐ |
| 相邻比较      | 存在不动元素（如大面积黑块），动态元素改变位置即可，无需分析具体轨迹                  | 易与位置规律混淆，关键区分是否有元素保持不动      | ⭐⭐⭐⭐ |
| 旋转翻转      | 旋转分顺时针 / 逆时针（常见 45° / 90° / 180°），翻转分左右 / 上下（对称轴变化） | 箭头法 / 时针法快速判定分组，需保持作图标准一致   | ⭐⭐   |
| 九宫格规律     | 等差、恒定、周期三种出题偏好，横向为主，纵向和米字型较少出现                      | 周期规律易被忽略，需注意交替变化            | ⭐⭐   |
| 16 宫格高频考法 | 内部相等时内外分开分析运动轨迹，近年创新递增规律（如蜗牛运动轨迹保留）                 | 内部递增是 25 年新趋势，需重点掌握         | ⭐⭐⭐⭐ |
| 25 宫格与蜂窝  | 斜向运动为主，整体运动需通过大面积区域定位，边缘地带优先观察                      | 斜向运动空间大，难度高于 16 宫格          | ⭐⭐⭐  |
| 非宫格类难点    | 隐藏线条（三角形 / 正方形 / 菱形框架）或拆分多部分运动，需就近原则找特殊             | 线条类题目需区分端点、焦点、十字交点          | ⭐⭐⭐⭐ |
| 解题技巧      | 大面积优先、个数统计、就近假设、拆分思维、对比选项找差异                        | 动态题先分析不动部分再验证变化部分           | ⭐⭐⭐  |
| 心理建设      | 允许出错，难题需标记后跳，图推每年创新，理解出题逻辑比刷题更重要                    | 70% 正确率即为高分，避免焦虑            | ⭐    |

**使用顺序建议**：
先判是否为宫格 → 再找不动元素 → 最后验证运动方式

# 样式规律 · 总览速览表

| 知识点   | 核心内容                       | 考试重点 / 易混淆点                       | 难度系数  |
| ----- | -------------------------- | --------------------------------- | ----- |
| 样式规律  | 部分相同元素的运算关系（加 / 减 / 同 / 异） | 区分相加 / 相减 / 求同 / 求异，**90% 题目考求异** | ⭐⭐⭐⭐  |
| 黑白运算  | 外框相同、黑块数量不同的运算规则           | **相同区域秒杀法** vs 传统运算公式             | ⭐⭐    |
| 标记考点  | 点 / 线 / 角 / 面的特殊标记规律       | 标记线位置（江苏高频）、多个标记物关系               | ⭐⭐⭐⭐  |
| 动态样式  | 图形旋转后的运算处理                 | 前两图对不上需旋转<br>一三同则二转 / 二三同则一转      | ⭐⭐⭐⭐⭐ |
| 拆分思维  | 不同区域采用不同运算规则               | 上海题常见：上下部分分别求同求异                  | ⭐⭐⭐⭐  |
| 金字塔运算 | 多层级递减式黑白运算                 | 上海特色题：下层两个元素决定上层一个                | ⭐⭐⭐⭐⭐ |

# 属性规律 · 总览速览表

| 知识点   | 核心内容                                | 考试重点 / 易混淆点               | 难度系数  |
| ----- | ----------------------------------- | ------------------------- | ----- |
| 对称性   | 优先考虑对称轴方向、数量及与线 / 面 / 点的关系          | 区分轴对称、中心对称、轴 + 中心对称图形     | ⭐⭐⭐⭐  |
| 曲直性   | 通过 1/2 或 2/3 比例定位（纯曲 / 纯直 / 曲直混合）   | 联考可能不严谨，需结合有无曲线判断         | ⭐⭐    |
| 开闭性   | 生活化图形优先考虑，分全开 / 半开 / 全闭             | 哑铃状图形属半开半分（连接处为点）         | ⭐⭐    |
| 图形间关系 | 简单图形考相交（点 / 线 / 面），复杂图形考共同 + 复杂部分关联 | 订书机题：考首尾线与中间线位置关系         | ⭐⭐⭐⭐  |
| 黑白块专项 | 对称性 ＞ 位置规律 ＞ 黑白运算；1 黑 + 多白优先图形关系    | 黑面与外框线数量、白面连通性、黑白面相交线数    | ⭐⭐⭐⭐  |
| 最新趋势  | 浙江题考未相交部分形状，国考题考相交线位置标记             | 母题变形（如碰外框 → 碰 1 条 / 2 条线） | ⭐⭐⭐⭐⭐ |
