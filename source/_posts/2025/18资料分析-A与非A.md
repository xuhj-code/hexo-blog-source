# A与非A · 核心速览
| 现期比重          | 比重比较            | 比重差          |
| ------------- | --------------- | ------------ |
| a ÷ b，先判“谁占谁” | 分子大 / 分母小 → 比重大 | 同总体：部分差 ÷ 总体 |
| “A占B”≠“B中A”   | 不同总体先算比重        | 不同总体：先算再相减   |
| 单位口径必须统一      | 倍数 / 直除 / 倒数    | 方向先判，大小后算    |

| 基期比重             | 两期比重变化        | 多期 / 图表      |
| ---------------- | ------------- | ------------ |
| 现期比重×(1+b)/(1+a) | a＞b 上升，a＜b 下降 | 连续多年定首尾      |
| 增速带正负号           | 可逆向推增长率       | 饼图先看 1/2、1/3 |
| 算前先判＞1 / ＜1      | 先方向后计算        | 非常规排序必核      |

| 比重连乘      | 利润率 / 产销率   | 高频陷阱     |
| --------- | ----------- | -------- |
| 小占中 × 中占大 | 利润/收入；销量/产量 | 比重 ≠ 增长率 |
| 过渡项可消元    | 区分统计口径      | 营业额 ≠ 数量 |
| 链式比例      | 零增长特例       | 单位不统一    |

# A与非A · 小结总览
| 知识点     | 核心内容                             | 考试重点 / 易混淆点                          | 难度系数 |
| ------- | -------------------------------- | ------------------------------------ | ---- |
| 比重问题关键字 | “占”字识别，特殊表述如利润率、增长贡献率等           | 现期比重计算与比较，特定比重计算步骤                   | ⭐⭐⭐  |
| 现期比重计算  | a 占 b 的比重 = a ÷ b；“在什么当中占比”需明确总体 | 分数比较注重范围认知，小除大不好除时用大除小看倒数            | ⭐⭐   |
| 特定比重计算  | 比重大于某值的计算：总数 × 比重 → 比较           | 注意避免将全国数据误算在内                        | ⭐⭐   |
| 现期比重差   | 相同总体用部分差 / 总体；不同总体分别算比重再做差       | 计算量通常不大，注意总体识别                       | ⭐⭐⭐  |
| 比重比较    | 从低到高需标号，范围认知优先                   | 分子大分母小则值大，可倒着比较                      | ⭐⭐   |
| 基期比重    | 给现期和A与非A：高减低加；给现期和增长率：公式计算        | 非新手算前看后，计算型选手根据差距选择约分方式              | ⭐⭐⭐⭐ |
| 两期比重比较  | 升降判断：a > b 上升，a < b 下降           | 连续多年比较定首尾看内部，逆运用通过比重变化推增长率           | ⭐⭐⭐  |
| 两期比重差计算 | 公式计算，先判升降再计算                     | 注意 a − b 写成 0.x 形式，1 + a 取 1.x 或 0.x | ⭐⭐⭐⭐ |
| 综合分析技巧  | 优先看分析性选项（两期比例比较、混合比例等）           | 表述绝对的选项易错，委婉表述正确率高                   | ⭐⭐⭐⭐ |
| 计算优化    | 一步除法看选项差距决定列式方式                  | 数字敏感度培养，勇于承担合理计算风险                   | ⭐⭐⭐⭐ |
