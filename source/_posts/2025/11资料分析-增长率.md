# 增长率 · 核心速览
| 增长率定义                   | 增长率 ↔ 百分点                            | 特定增长率判断                                              |
| ----------------------- | ------------------------------------ | ---------------------------------------------------- |
| 增长率 = 增长量 / 基期量；又称增速/增幅 | 提高/降低X百分点 → 直接加减；降幅扩大/收窄 → 先算绝对值再加负号 | >10%：现期 > 基期×1.1；>50%：现期 > 基期×1.5；降幅>10%：现期 < 基期×0.9 |

| 增长率比较技巧                                   | 年均增长率变形题                 | 材料陷阱与速算                                  |
| ----------------------------------------- | ------------------------ | ---------------------------------------- |
| 倍数明显直接比现期/基期；不明显用增长量/基期量；柱状图高度差占比大 → 增长率大 | 题干问“年均增长率高于全社会X%” → 直接加减 | 累计数据验证历史年份；月均相同 → 全年=前几月×倍数；排序题用排除法+最值思维 |

| 核心公式抓手         | 快速估算               | 比较策略               |
| -------------- | ------------------ | ------------------ |
| 增长率 = Δ现期 / 基期 | 错位加减法（如15%拆10%+5%） | 柱状/折线图对比 → 高低差占比判断 |

# 增长率 · 小结总览
| 知识点        | 核心内容                                                                          | 考试重点 / 易混淆点                  | 难度系数 |
| ---------- | ----------------------------------------------------------------------------- | ---------------------------- | ---- |
| 增长率定义与公式   | 增长率 = 增长量 / 基期量；又称增速、增幅、增长幅度等                                                 | 区分“增长倍数”与“倍数”（倍数 = 增长率 + 1）  | ⭐⭐   |
| 增长率与百分点的关系 | 增速较上年“提高/降低”X个百分点：直接加减（带正负号）；降幅“扩大/收窄”X个百分点：先算绝对值再整体加负号                       | 降幅比较时需看绝对值（如-40%降幅 > -20%降幅） | ⭐⭐⭐  |
| 特定增长率快速判断  | >10%：现期 > 基期×1.1（错位相加）；>50%：现期 > 基期×1.5（加一半）；降幅>10%：现期 < 基期×0.9 或 基期 > 现期×1.1 | 15%拆解为10%+5%（需精确计算）          | ⭐⭐⭐⭐ |
| 增长率比较技巧    | 倍数明显时直接比现期/基期；不明显时算增长量/基期量；给现期和增长量时可用增长量/现期量代替比较                              | 柱状图高度差占比越大增长率越大              | ⭐⭐⭐  |
| 年均增长率变形题   | 题干问“年均增长率高于全社会1.2%”即城镇 = 全社会1.5% + 1.2% = 2.7%                                | 注意非“高减低加”，而是直接加减关系           | ⭐⭐⭐⭐ |
| 材料陷阱与速算    | 累计数据需验证是否包含历史年份；月均相同则“全年=1-4月×3”；选项含“约”字慎用尾数法；排序题用排除法+最值思维                    | -                            | ⭐⭐⭐⭐ |
