# 现期量 · 核心速览
| 现期量公式         | 限期追赶      | 小追大（追得上/追不上）     |
| ------------- | --------- | ---------------- |
| 增量保持用加，增率保持用乘 | 代入法快速判断年份 | 补差距 / 加差距 → 年限计算 |

| 大追小（倍数目标）         | 名义 vs 实际                | 基期价计算       |
| ----------------- | ----------------------- | ----------- |
| 现期量=基期量×(1+增长率)^n | 名义=实际+CPI+交叉项，名义>实际→物价涨 | 用实际增长率反推基期量 |

| 利滚利思想             | 核心策略     | 快速判断         |
| ----------------- | -------- | ------------ |
| 拆分本金与利息，大额精确，小额估算 | 比例关系快速判断 | 先抓公式，关键差距/倍数 |

📌 用法说明

考前翻页 → 一眼抓公式/追赶/倍数/利滚利

做题中遇到复杂追赶或增长题 → 直接对照表格思路

# 现期量 · 小结总览

| 知识点          | 核心内容                                            | 考试重点 / 易混淆点             | 难度系数  |
| ------------ | ----------------------------------------------- | ----------------------- | ----- |
| 现期量计算        | 现期量=基期量+增长量（保持增量不变）<br>现期量=基期量×(1+增长率)（保持增长率不变） | 区分“保持增量”与“保持增长率”的公式差异   | ⭐⭐⭐   |
| 限期追赶（定量）     | 通过代入法快速判断哪一年超过目标值（如150万件）                       | 临界值估算的精确性要求             | ⭐⭐⭐⭐  |
| 小追大（追得上）     | 找差距补差距：初始差距÷（追赶方增量-被追赶方增量）=所需年限                 | 增量比较与年份差计算              | ⭐⭐⭐⭐  |
| 小追大（追不上）     | 找差距加差距：初始差距+（追赶方增量-被追赶方增量）×年限=最终差距              | 基数与增量双重劣势的判定            | ⭐⭐⭐⭐  |
| 大追小（倍数目标）    | 现期量=基期量×(1+增长率)^n，通过倍数关系或代入法求解                  | 高次幂计算的简化技巧（如平方数拆分）      | ⭐⭐⭐⭐⭐ |
| 名义 vs 实际增长率  | 名义增长率=实际增长率+CPI增长率+交叉项                          | 名义>实际→物价上涨；公式变形与选项代入排除法 | ⭐⭐⭐   |
| 按基期价格计算现期量   | 现期量=基期量×(1+实际增长率)                               | 实际增长率应用场景的理解            | ⭐⭐⭐⭐  |
| 利滚利思想（增长量估算） | 拆分本金与利息：大额部分精确计算，小额部分按比例估算                      | 比例关系的快速判断（如12倍近似）       | ⭐⭐⭐⭐⭐ |
